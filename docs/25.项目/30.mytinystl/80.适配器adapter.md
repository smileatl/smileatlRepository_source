# 配接器

主要用来修改接口。就像转接口一样, 将一个对class封装变成了另外一个功能的class。

STL 主要提供如下三种配接器：

1.  改变仿函数（functors）接口，称之为 function adapter
2.  改变容器（containers）接口，称之位 container adapter
3.  改变迭代器（iterators）接口，称之为 iterator adapter
