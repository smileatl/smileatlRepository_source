![04_信号](/assets/klszjiog1u9345h.png)

## 终端按键产生信号

​    Ctrl + c  → 2) SIGINT（终止/中断）   "INT" ----Interrupt

​    Ctrl + z  → 20) SIGTSTP（暂停/停止）  "T" ----Terminal 终端。

​    Ctrl + \  → 3) SIGQUIT（退出） 



真正执行信号捕捉动作的是内核
